Brick Breaker – Turtle Edition\
by @MarkChaitra

Description:\
This is a Brick Breaker (also known as Breakout) game created using Python’s turtle graphics library.\
The goal is to control the paddle at the bottom of the screen to bounce the ball and break all the colored bricks.

How to Play:\
Use A to move the paddle left.\
Use D to move the paddle right.\
The ball bounces off walls, the paddle, and bricks.\
Each brick changes color as it’s hit:\
Red → Blue → White → Broken (disappears)\
Once all 18 bricks are broken, the game resets automatically.\
If the ball falls below the paddle, the game also resets.\

Game Features:\
Real-time paddle control with smooth movement.\
Progressive brick damage (each needs 3 hits to break).\
Dynamic ball collision with edges, paddle, and bricks.\
Automatic reset after win or loss.\
Simple and clean Turtle graphics layout.\
