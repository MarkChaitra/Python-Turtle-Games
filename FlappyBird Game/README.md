Flappy Bird – Turtle Edition\
by @MarkChaitra

Description:\
This project is a simple version of the classic Flappy Bird game built using Python’s turtle graphics library.\
You control a small yellow bird that must fly between moving green pipes without hitting them or falling off the screen.\

How to Play:
Press the Spacebar to make the bird flap upward.\
The bird will continuously fall due to gravity, so time your jumps carefully.\
Avoid hitting the top or bottom pipes and the screen boundaries.\
Each time you successfully pass a pair of pipes, your score increases by 1.\
If you hit a pipe or the edge of the screen, the game resets after a short delay.

Game Features:
Smooth gravity and flap mechanics.\
Randomly generated pipes to create continuous gameplay.\
Real-time score tracking with a high score display.\
Automatic reset when the player loses.\

Controls:\
Spacebar: Flap / make the bird go up.
